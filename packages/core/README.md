# @paystream/core

**PayStream x402 protocol types, constants, and utilities for Bitcoin-native micropayments on Stacks.**

## Installation

```bash
npm install @paystream/core
```

## Features

- Shared types for x402 payment protocol.
- Constants for Stacks networks and token addresses.
- Validation utilities for payment receipts and requests.
- Encoding/decoding for x402 headers.
